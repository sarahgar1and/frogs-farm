About:

How to run: 
Run the file "main" in editor. The other python files "mapGeneration", "classes", 
"cheatKeys", and "startData" should be in the same folder. Additionally,
a have a folder called "images" in the same location as "main" which contains all 
images needed (images found in google drive). The "data" csv file should initially be 10 
lines long, where the first 4 lines are "0 20 20 20" and the rest are empty.

The only library you need is cmu_graphics!

Shortcuts: 
press 'h' in farm area to immediately make all plants harvestable.
press 'm' in farm area to add 100 coins to your money.
