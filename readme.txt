Welcome to Frog's Farm!

About:
This project is a farming simulator with a farming buddy: Frog! Make sure
to Frog gets enough sleep and food. With a well cared for frog and plants,
your agrarian empire is sure to grow!

How to run: 
Run the file "main.py" in editor. The other python files "mapGeneration.py", 
"classes.py", "cheatKeys.py", and "startData.py" should be in the same folder. 
Additionally, a have a folder called "images" in the same location as "main" 
which contains all images needed (images found in google drive). The "data" csv 
file should initially be 10 lines long, where the first 4 lines are "0 20 20 20" 
and the rest are empty.

The only library you need is cmu_graphics!

Shortcuts: 
press 'h' in farm area to immediately make all plants harvestable.
press 'm' in farm area to add 100 coins to your money.
